Personal projects

Those projects are my favorite. Especially the Hotel reservation application and Gitlet, which are build from scracth. Please have a look at them!

Hotel Reservation App
  • Created a command line App with two menus designed for reservation and administration.
  • Supported account creation, availability checking, auto recommendation, room adding, etc.
  • Structuralized the App with four layers to separate responsibilities and support modularization.

Parallel WebCrawler
  • Improved a sequential web crawler to a parallel web crawler using fork-join pools and synchronization.
  • Restructured the project using maven and utilized several design patterns to improve the reusability of the project.
  • Created a profiler to record the performance of the web crawler. Improve the performance nearly 3 times.

NumC
  • Mimicked NumPy library in C including matrix multiplication, power of a matrix, and several simple operations.
  • Combined repeated squaring algorithm, SIMD instructions, and OpenMP library in implementations.
  • Speeded up matrix multiplication around 56 times and power of a matrix around 1200 times.

Gitlet
  • Implemented a version-control system from scratch that mimics some of the basic features of Git.
  • Supporting nearly 20 commands including merge, push, pull, local remote, etc.
  • Boosted merging speed by using the Lowest Common Ancestor algorithm.

Ataxx
  • Built a board game from concept to product, accomplished with an opposing AI.
  • Utilized alpha-beta pruning, heuristic estimation, and minimax algorithm to implement the AI.
  • Accomplished the AI with the ability to guarantee a win within a certain number of moves.
